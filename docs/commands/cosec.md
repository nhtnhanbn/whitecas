cosec(*θ*)

- *θ* is an angle

Returns the cosecant of *θ*.

### Example

Find the largest set of values of $$ x $$ for which $$ \cos(x) > \dfrac{1}{4} \csc(x) $$. (2017 Specialist Exam 2 Section A Question 2)
