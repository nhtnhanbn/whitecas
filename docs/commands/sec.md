sec(*θ*)

- *θ* is an angle

Returns the secant of *θ*.

### Example

Find the expression for $$ \int_0^{\tfrac{\pi}{6}} \tan^2(x)\sec^2(x) dx $$ following a suitable substitution. (2018 Specialist Exam 2 Section A Question 8)
